HTML Tags Documentation

This page explains all the HTML tags used in the Personal Debt Tracker application.

<!DOCTYPE html>

Defines the document type and HTML version (HTML5 in this case).

<html>

The root element of an HTML page. The lang="en" attribute specifies that the document is in English.

<head>

Contains meta-information about the document, including:
  • <meta charset="UTF-8"> - Specifies character encoding
  • <meta name="viewport"> - Controls mobile responsiveness
  • <title> - Sets the browser tab title
  • <link rel="stylesheet"> - Links to external CSS files

<body>

Contains all the visible content of the web page.

<div>

A container element used to group other elements for styling and organization. Key divs in the app:
  • header - Contains the logo and title
  • dashboard - Holds the summary cards
  • card - Individual summary cards
  • form-container - Wraps the debt entry form
  • form-group - Groups form labels and inputs

<img>

Displays an image. Attributes:
  • src - Path to the image file
  • alt - Alternative text for accessibility

<h1>, <h2>, <h3>

Heading elements of different levels (h1 is most important). Used for:
  • Page title (h1)
  • Section headings like "Debts Owed to Me" (h2)
  • Card titles like "Total Owed to Me" (h3)

<table>

Creates a data table with rows and columns. Contains:
  • <thead> - Table header with column labels
  • <tbody> - Table body where debt data is dynamically inserted
  • <tr> - Table row
  • <th> - Table header cell
  • <td> - Table data cell

<form>

Container for user input elements. Attributes:
  • id="debt-form" - Unique identifier for JavaScript
Contains various form controls:
  • <input> - For text, number, and date entry
  • <select> - Dropdown for debt type selection
  • <button> - Form submission and reset buttons
  • <label> - Descriptions for form fields

<script>

Links to external JavaScript file that provides all the interactive functionality:
  • Adding and deleting debts
  • Calculating totals
  • Updating the display
  • Form handling
